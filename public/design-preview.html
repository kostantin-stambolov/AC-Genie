Design System Comparison

A — Slate Pro

Modern SaaS. Circle steppers, subtle borders, clean type.

✓ Legible, professional

✓ Closest to current

B — Deep Structure

Dark nav bar, left-border accents, progress bar.

✓ Authoritative / trustworthy

✓ Highest info density

C — Open Academy

Stripe-inspired. Float cards, lift-hover CTAs.

✓ Premium feel

✓ Very approachable

D — Social Flow ✦

Duolingo-like. Story rings, streak, stat cards, engaging.

✓ Highest engagement

✓ Gamified progress

E — Editorial ✦

NYT-inspired. Serif headings, rules, §section numbers.

✓ Maximum authority

✓ Very distinctive

Icon Set Recommendation

All three options use Lucide Icons (already available as lucide-react in your Next.js project). Stroke weight: 1.75px. Key icons used: pen-line, compass, zap, sparkles, graduation-cap, arrow-right, check-circle-2, lightbulb, history, route.
